Частые вопросы

Бесплатно ли это?

Да, базовые функции SellSync бесплатны: оба маркетплейса (WB и Ozon), все типы данных (заказы, продажи, остатки, финансы), юнит-экономика, ABC-анализ и прогноз поставок. Подписка появится позже и закроет авто-обновление и расширенную историю.

Где взять API-токен Wildberries?

В личном кабинете продавца WB: Профиль → Настройки → Доступ к API → Создать новый токен. Отметьте категории Контент, Маркетплейс, Статистика, Финансы, Аналитика. Токен начинается с eyJ — скопируйте его сразу, после закрытия окна WB больше не покажет.

Где взять Client ID и API Key Ozon?

В кабинете Ozon Seller: Настройки → Seller API → Сгенерировать ключ. Выберите роль Admin Read Only — этого достаточно. Скопируйте API Key (UUID) и Client ID — оба значения нужно вставить в sidebar SellSync.

Как часто данные обновляются?

В Free-плане — вручную: нажимаете «Обновить сейчас» в sidebar, и SellSync синхронизирует все листы за 1–3 минуты. Авто-обновление по расписанию (раз в час, 6 или 12 часов) появится в платном плане.

Безопасно ли передавать API-токены через add-on?

Токены хранятся в UserProperties — это защищённое хранилище Google Apps Script, доступное только вам. SellSync не отправляет токены никуда, кроме API Wildberries и Ozon. Add-on запрашивает доступ только к текущей таблице (currentonly), а не ко всем вашим документам Google.

Чем SellSync отличается от WB-Sheets и Excellen?

WB-Sheets от AliRadar поддерживает только Wildberries и не обновлялся с февраля 2022. Excellen — это платный шаблон Google Sheets за 4 990 ₽, не нативный аддон, без авто-синхронизации. SellSync — единственный бесплатный нативный аддон в Google Workspace Marketplace, который покрывает и WB, и Ozon в одной таблице с юнит-экономикой и ABC-анализом.

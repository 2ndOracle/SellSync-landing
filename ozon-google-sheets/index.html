Как настроить выгрузку Ozon

Где взять Client ID и API Key Ozon — подробная инструкция.

Шаг 1

Установите add-on

Откройте Google Sheets → Расширения → Дополнения → SellSync. Один клик из Google Workspace Marketplace.

Шаг 2

Вставьте Client ID и API Key

В кабинете Ozon Seller: Настройки → Seller API → Сгенерировать ключ. Скопируйте Client ID и API Key, вставьте в sidebar SellSync.

Шаг 3

Синхронизируйте

Нажмите кнопку — за секунды появятся листы Ozon Заказы, Ozon Продажи, Ozon Остатки, Ozon Финансы и листы аналитики.

Ozon в Google Workspace Marketplace — почему это редкость

Для Wildberries в Google Workspace Marketplace есть аддон WB-Sheets (заброшен с 2022 года). Для Ozon до SellSync не было ни одного нативного решения — продавцы Ozon выгружали данные через MPStats, Albato или собственные скрипты. Несколько технических деталей, которые отличают SellSync на Ozon:

  1. FBO и FBS — оба формата отгрузок. Заказы тянутся через /v3/posting/fbs/list и аналогичный FBO endpoint, продажи и выручка — через analytics API. Лист «Ozon Заказы» содержит и собственный склад, и отгрузки с FBO.
  2. Комиссии и логистика приходят отдельно. В /v1/analytics/data финансовых полей нет — SellSync дополнительно опрашивает /v1/finance/transaction/list и обогащает листы «Ozon Продажи» и «Ozon Финансы» детальными данными по комиссии, эквайрингу, сервисным сборам и возвратам.
  3. Премиум-аналитика Ozon недоступна без подписки. Часть отчётов (рейтинг продавца, конверсии в выкуп, brand-аналитика) Ozon отдаёт только продавцам с Premium. Это ограничение платформы — SellSync обойти его не может, но базовые данные (заказы, продажи, остатки, финансы) работают на любом тарифе.
  4. Авторизация через два значения. Ozon Seller API требует Client ID (номер кабинета) и API Key (UUID-секрет). Достаточно роли «Admin Read Only» — SellSync только читает данные, ничего не пишет в кабинет.
  5. Остатки на разных складах. Лист «Ozon Остатки» собирает данные по всем складам Ozon отдельной строкой на склад — видно, где товар лежит и сколько в пути.

Частые вопросы про Ozon

Где взять Client ID и API Key Ozon?

В кабинете продавца на seller.ozon.ru: Настройки → Seller API → Сгенерировать ключ. Скопируйте API Key (UUID вида xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx) — после закрытия окна его больше не показывают. Client ID — короткий числовой идентификатор кабинета, отображается на той же странице.

Какую роль выбрать для API-ключа Ozon?

Достаточно роли Admin Read Only. SellSync только читает данные из кабинета — заказы, продажи, остатки, финансы. В кабинет Ozon ничего не записывается, поэтому права на запись не нужны.

Поддерживается ли FBO и FBS Ozon?

Да, оба формата отгрузок. Заказы выгружаются и для FBS (отгрузка со склада продавца), и для FBO (склады Ozon). В листе «Ozon Заказы» отгрузки видны вместе с признаком склада, по которому можно фильтровать.

Что с премиум-аналитикой Ozon?

Часть продвинутых отчётов Ozon (рейтинг продавца, конверсия в выкуп, brand-аналитика) доступна только продавцам с подпиской Premium. Это ограничение платформы — SellSync не может его обойти. Базовые данные — заказы, продажи, остатки, финансы — работают на любом тарифе Ozon.

Можно ли подключить несколько кабинетов Ozon?

В одной Google-таблице — одна пара Client ID + API Key Ozon. Если у вас несколько кабинетов, создайте отдельные Google-таблицы и установите SellSync в каждой со своими credentials.

Заказы, продажи, остатки и финансы Ozon — выгружаются прямо в вашу Google-таблицу. Бесплатно, без программирования, через Google Workspace Add-on.

Установить бесплатно

Бесплатно. Без регистрации. Установка за 1 минуту.

Какие данные Ozon выгружаются

Четыре листа в вашей Google-таблице, обновляемых по кнопке.

Заказы

Дата, номер, артикул, название, количество, цена, сумма, статус, склад. По каждому заказу из Ozon Seller API.

Продажи

Дата, артикул, выручка, комиссия, логистика, к выплате. Реальные продажи с разбивкой по товарам и периодам.

Остатки

Артикул, склад, доступно, в пути, всего. Текущие остатки по всем складам Ozon.

Финансы

Дата, тип операции, выручка, комиссия, логистика, возврат комиссии, сервисные сборы, итого. Источник — /v1/finance/transaction/list.

Аналитика поверх данных Ozon

Отдельные листы с формулами, пересчитываются после каждой синхронизации. Free tier.

Юнит-экономика

Маржа, ROI и валовая прибыль по каждому SKU. SUMIF по продажам, VLOOKUP по себестоимости — всё видно в одной таблице.

ABC-анализ

Категории A/B/C по доле в выручке. Видно сразу, какие товары делают прибыль, а какие тянут вниз ассортимент.

Прогноз поставок

Средние дневные продажи (7/14 дней), дни запаса, рекомендация к закупке. Conditional formatting подсвечивает критичные позиции.

Себестоимость

Лист с ручным вводом закупочных цен. Цены сохраняются при обновлении и подтягиваются в юнит-экономику автоматически.

Как настроить выгрузку Ozon

Шаг 1

Установите add-on

Откройте Google Sheets → Расширения → Дополнения → SellSync. Один клик из Google Workspace Marketplace.

Шаг 2

Вставьте Client ID и API Key

В кабинете Ozon Seller: Настройки → Seller API → Сгенерировать ключ. Скопируйте Client ID и API Key, вставьте в sidebar SellSync.

Шаг 3

Синхронизируйте

Нажмите кнопку — за секунды появятся листы Ozon Заказы, Ozon Продажи, Ozon Остатки, Ozon Финансы и листы аналитики.

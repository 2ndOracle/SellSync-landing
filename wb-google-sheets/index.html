Особенности API Wildberries — что важно знать

SellSync ходит к нескольким endpoint'ам WB Statistics API и склеивает данные в листы. Несколько нюансов, на которые продавцы натыкаются чаще всего:

  1. Поле «К выплате» (forPay) считается асинхронно. Первые 24 часа после продажи WB возвращает 0. Точные суммы приходят позже — и параллельно дублируются в листе «WB Финансы» из отчёта reportDetailByPeriod.
  2. Комиссия и логистика в /api/v1/supplier/sales отсутствуют. SellSync берёт их из детального финансового отчёта reportDetailByPeriod и распределяет пропорционально выручке. Если расхождение принципиально — смотрите лист «WB Финансы», там данные напрямую с финансового endpoint'а.
  3. Endpoint остатков /api/v1/supplier/stocks помечен на удаление в changelog WB. SellSync мониторит обновления и переключится на новый endpoint автоматически — пользователю ничего не надо менять.
  4. Штрафы и удержания. Все списания (упаковка, габариты, отмены) собираются в листе «WB Финансы» с типом операции — видно, за что и сколько ушло.
  5. Категории доступа в токене. Для полной выгрузки нужны Контент, Маркетплейс, Статистика, Финансы, Аналитика. Без любой из них часть листов будет пустой.

Частые вопросы про Wildberries

Где создать API-токен Wildberries?

В личном кабинете продавца на seller.wildberries.ru: нажмите на профиль → Настройки → Доступ к API → Создать новый токен. Назовите токен SellSync, выберите нужные категории доступа и сохраните длинную строку, начинающуюся с eyJ — после закрытия окна WB её больше не покажет.

Какие категории доступа выбрать в WB при создании токена?

Для полной выгрузки нужны пять категорий: Контент, Маркетплейс, Статистика, Финансы, Аналитика. Без любой из них часть листов будет пустой. Опцию «Только для чтения» можно оставить включённой — SellSync не пишет в кабинет WB.

Чем SellSync отличается от WB-Sheets от AliRadar?

WB-Sheets — единственный другой нативный аддон для Wildberries в Google Workspace Marketplace, но он поддерживает только WB (без Ozon), не обновлялся с февраля 2022 года и запрашивает полный доступ ко всем Google-таблицам пользователя. SellSync покрывает WB и Ozon одновременно, активно поддерживается, использует минимальные права (currentonly — только текущая таблица) и добавляет юнит-экономику, ABC-анализ и прогноз поставок поверх данных.

Почему «К выплате» в листе «WB Продажи» равно 0?

Wildberries рассчитывает поле forPay асинхронно — в первые 24 часа после продажи там может быть 0. Точная сумма к выплате появится позже автоматически. Параллельно её можно посмотреть в листе «WB Финансы», где данные приходят напрямую из отчёта reportDetailByPeriod.

Можно ли подключить несколько кабинетов Wildberries?

В одной Google-таблице — один токен WB. Если у вас несколько кабинетов, создайте отдельные таблицы Google и установите SellSync в каждой со своим токеном.
